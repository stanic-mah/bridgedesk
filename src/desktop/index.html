How to use BridgeDesk

              
Recommended setup for a fixed ChatGPT MCP URL.

            

            
          


          

            

              
1. Install Cloudflare Tunnel

              
Install the Cloudflare Tunnel client, then reopen BridgeDesk and select Refresh Checks.

              
winget install --id Cloudflare.cloudflared

              
            


            

              
2. Use Quick Tunnel

              
    
                
  1. Choose one project folder.
    2. 
                
  2. Select Quick mode.
    3. 
                
  3. Select Save Config, then Start Tunnel.
    4. 
                
  4. Wait for a trycloudflare.com URL, then select Start Server.
    5. 
                
  5. Copy the MCP URL into ChatGPT. If the Quick URL changes later, update or recreate the ChatGPT app connection.
    6. 
              

            


            

              
3. Use Permanent Tunnel

              
    
                
  1. Choose one project folder. Do not choose the whole drive or whole user folder.
    2. 
                
  2. Select Permanent mode.
    3. 
                
  3. Set Tunnel name to bridgedesk.
    4. 
                
  4. Enter the fixed public URL without /mcp, for example https://mcp.yourdomain.com.
    5. 
                
  5. Select Save Config, Cloudflare Login, Create Tunnel, Route DNS, Start Permanent Tunnel, then Start Server.
    6. 
              

            


            

              
4. Check Cloudflare Security

              

                Turn off Cloudflare Bot Fight Mode for the BridgeDesk domain. ChatGPT uses automated server requests,
                and Bot Fight Mode can block the OAuth connection after the owner password page.
              

              
Block AI bots can usually stay on. Turn it off only if ChatGPT still cannot connect.

            


            

              
5. Add BridgeDesk in ChatGPT

              
    
                
  1. Open ChatGPT Settings, then Apps.
    2. 
                
  2. Create a new app named BridgeDesk.
    3. 
                
  3. Use Server URL mode and paste the MCP URL from BridgeDesk.
    4. 
                
  4. Keep Authentication as OAuth.
    5. 
                
  5. Accept the custom MCP warning, then create or connect.
    6. 
                
  6. Paste the owner password from BridgeDesk when the approval page opens.
    7. 
              

            


            

              
6. Switch Project Folder

              
    
                
  1. Select Choose beside Project folder and pick the new folder.
    2. 
                
  2. Select Stop Server, then Start Server again.
    3. 
                
  3. In Permanent mode, the MCP URL stays the same. In Quick mode, update ChatGPT only if the tunnel URL changed.
    4. 
              

              

                If ChatGPT shows Stop thinking after switching folders, wait. It may ask you to reconnect BridgeDesk.
                Reconnect with the owner password and let the same run continue.
              

            


            

              
7. Daily Use

              

                Start Permanent Tunnel and Start Server before using ChatGPT. You can change the project folder without
                changing the MCP URL. Use Stop All when you want to shut down both BridgeDesk processes.
              

              
In ChatGPT, ask: Use BridgeDesk and open the selected project folder.

            


            

              
Troubleshooting

              
    
                
  • If ChatGPT fails after the owner password, disable Cloudflare Bot Fight Mode.
    • 
                
  • If the URL shows /mcp/mcp, remove /mcp from Fixed public URL and save again.
    • 
                
  • If ChatGPT connects but shows no actions, click Refresh in ChatGPT or recreate the app entry.
    • 
                
  • If port 7676 is busy, stop the old BridgeDesk server or choose another port.
